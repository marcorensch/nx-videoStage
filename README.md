# nx-videostage
Special HTML5 Video Player for Joomla!

![Hires Demo](http://nx-productions.ch/docs/nx-videostage/img/hires-demo.png)

## Features

### Three Player Modes
- Ambilight Player
- Drop Shadow Player
- Default Player

### Textblock
- Textblock Position
  - Left, Right, Top, Bottom
- Stage Background Color (rgba)

### Customizable Playback
- Autoplay
- Initial Volume
- Border
- Border-Radius
